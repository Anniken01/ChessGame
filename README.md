# Chess Game in Java

Implemented as part of CSCI 1933 - Introduction to Algorithms and Data Structures at the University of Minnesota.

A two-player chess game played in the terminal with all standard pieces and movement rules, including pawn promotion.

## How to run
```
javac CSCI1933P2/Game.java
java CSCI1933P2/Game
```

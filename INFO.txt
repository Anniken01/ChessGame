How to run and Compile:
Go to the folder CSCI1933P2 and run the line:
"javac CSCI1933P2/Game.java"
and then run
"java CSCI1933P2/Game"
The user will now be prompted to make a move, and the program will print out which users turn it is.

Assumptions:
We assume the user will enter valid coordinates, such as their numbers are in bounds and they do not type in any other characters.
To end the game (other than typing 'exit' directly), we assume the user will hit any key other than valid coordinates.
